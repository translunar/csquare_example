= CSquare example

http://github.com/mohawkjohn/csquare_example

== Description

This is an example for the csquare gem which includes rational, complex, and Ruby object types.

A derivative form will be included in NMatrix v0.0.2 for matrix operations.

== Usage

    gem install csquare

Then, in IRB, simply:

    require "./test.rb"

The output will be in test.c.
